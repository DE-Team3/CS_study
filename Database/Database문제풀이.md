**Q. B-tree란?**    
<details>
<summary>정답</summary>
<br>
 인덱스 자료구조를 표현하는 방식중 하나이다 
 B-Tree는 노드하나가 여러 데이터를 저장한다 각 노드별로 여러개의 키와 키에 대응하는 데이터를 품고있으며 정렬된 상태를 유지하기에 부등호 표현식(연산)이 가능하고 
 데이터 탐색뿐 아니라 저장 삭세 수정 삽입등에도 낮은 시간 복잡도를 가진다
</details>
